* 所有需要对用户展示文本的地方使用如下方式进行翻译，导入`ugettext_lazy`函数为`_`，接收一个参数，为原始文本对应的英文字符串:

```
from django.utils.translation import ugettext_lazy as _

# Some message to show
msg = _('This message will be translated later')
```
